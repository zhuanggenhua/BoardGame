## 📝 翻译原则

1. **语义优先**: 基于音效实际含义翻译，而非逐词翻译
2. **分类前缀**: 使用中文分类前缀（如"咒语·"、"元素·"、"界面·"）
3. **保留数字后缀**: 如 "骨骼断裂 01"、"拳击命中 02"
4. **一致性**: 同类音效使用统一的翻译模式
