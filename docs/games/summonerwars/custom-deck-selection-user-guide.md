# 召唤师战争 - 自定义牌组选择功能使用指南

## 功能简介

自定义牌组选择功能允许你在阵营选择界面直接选择和使用你预先构建的自定义牌组，无需每次都重新构建。你可以保存最多 2 个自定义牌组，并在对局开始前快速选择。

## 功能特点

- 🎴 **快速选择** - 在阵营选择界面直接选择已保存的自定义牌组
- ✏️ **即时编辑** - Hover 牌组卡片即可快速编辑
- 🆕 **快速创建** - 点击"+"按钮快速创建新牌组
- 💾 **自动保存** - 牌组修改后自动保存到云端
- 🎨 **视觉区分** - DIY 徽章清晰标识自定义牌组

## 使用前提

- ✅ 已注册并登录账号
- ✅ 已使用牌组构建器创建并保存至少一个自定义牌组

## 如何使用

### 1. 查看已保存的自定义牌组

**步骤**:
1. 登录你的账号
2. 创建或加入一个召唤师战争对局
3. 在阵营选择界面，你会看到：
   - 6 个默认阵营卡片
   - 你保存的自定义牌组卡片（最多 2 个）
   - "+"按钮（如果自定义牌组少于 2 个）

**识别自定义牌组**:
- 左上角有紫色 **DIY** 徽章
- 显示牌组名称
- 显示召唤师精灵图

### 2. 选择自定义牌组

**步骤**:
1. 在阵营选择界面，点击你想使用的自定义牌组卡片
2. 卡片会显示金色边框，表示已选中
3. 玩家状态区会显示：
   - "自定义牌组"标签
   - 紫色 DIY 徽章
   - 召唤师所属阵营信息
4. 点击"准备"或"开始游戏"按钮

**注意事项**:
- 选择自定义牌组后，游戏将使用你保存的牌组配置
- 确保你的牌组符合游戏规则（召唤师 + 至少 20 张卡牌）

### 3. 编辑已保存的自定义牌组

**步骤**:
1. 在阵营选择界面，将鼠标悬停在自定义牌组卡片上
2. 右上角会出现编辑按钮（铅笔图标）
3. 点击编辑按钮
4. 牌组构建器会从底部滑出，并加载该牌组的数据
5. 修改牌组内容（添加/删除卡牌）
6. 点击"保存"按钮
7. 牌组构建器关闭，阵营选择界面自动刷新

**提示**:
- 编辑按钮只在 Hover 时显示，避免误触
- 修改会立即保存到云端
- 如果其他玩家正在占用该牌组，你仍然可以编辑

### 4. 创建新的自定义牌组

**步骤**:
1. 在阵营选择界面，点击"+"按钮
2. 牌组构建器会从底部滑出，处于新建模式
3. 选择召唤师
4. 从卡牌池中添加卡牌到你的牌组
5. 输入牌组名称
6. 点击"保存"按钮
7. 牌组构建器关闭，阵营选择界面自动刷新，显示新创建的牌组

**限制**:
- 最多保存 2 个自定义牌组
- 如果已有 2 个牌组，"+"按钮不会显示
- 需要先删除一个现有牌组才能创建新牌组

### 5. 删除自定义牌组

**步骤**:
1. 按照"编辑已保存的自定义牌组"步骤打开牌组构建器
2. 在牌组构建器中找到"删除"按钮
3. 点击"删除"按钮
4. 确认删除操作
5. 牌组构建器关闭，阵营选择界面自动刷新

**注意**:
- 删除操作不可撤销
- 删除后，该牌组会从云端永久移除

## 常见问题

### Q1: 为什么我看不到自定义牌组？

**A**: 可能的原因：
1. 你还没有登录账号
2. 你还没有创建并保存任何自定义牌组
3. 网络连接问题，无法加载牌组列表

**解决方法**:
- 确保已登录账号
- 先使用牌组构建器创建并保存至少一个牌组
- 检查网络连接，刷新页面重试

### Q2: 为什么"+"按钮不显示？

**A**: 你已经保存了 2 个自定义牌组（最大数量）。需要先删除一个现有牌组才能创建新牌组。

### Q3: 如何知道我的牌组是否符合规则？

**A**: 牌组构建器会自动验证牌组规则：
- 必须选择一个召唤师
- 至少 20 张卡牌
- 不超过最大卡牌数量限制
- 每张卡牌不超过数量限制

如果牌组不符合规则，"保存"按钮会被禁用。

### Q4: 我可以在对局中修改牌组吗？

**A**: 不可以。牌组只能在阵营选择阶段修改。一旦游戏开始，牌组配置就固定了。

### Q5: 其他玩家能看到我的自定义牌组吗？

**A**: 在阵营选择界面，其他玩家可以看到你选择了"自定义牌组"，但看不到具体的牌组内容。游戏开始后，牌组内容会逐渐展示。

### Q6: 自定义牌组会同步到其他设备吗？

**A**: 是的。自定义牌组保存在云端，你可以在任何设备上登录账号并使用你的牌组。

### Q7: 如果加载牌组失败怎么办？

**A**: 系统会显示错误提示（Toast 通知）。你仍然可以选择默认阵营进行游戏。建议：
- 检查网络连接
- 刷新页面重试
- 如果问题持续，联系技术支持

## 快捷键

目前自定义牌组选择功能不支持快捷键。所有操作都通过鼠标点击完成。

## 最佳实践

### 1. 牌组命名
- 使用描述性的名称，如"凤凰精灵-快攻"、"冰原兽人-防守"
- 避免使用过长的名称（建议 10 个字符以内）

### 2. 牌组管理
- 定期更新牌组以适应游戏平衡调整
- 为不同的游戏风格创建不同的牌组
- 删除不再使用的牌组，为新牌组腾出空间

### 3. 对局准备
- 在对局开始前检查牌组是否符合当前版本的规则
- 确保牌组适合当前的对手和游戏模式
