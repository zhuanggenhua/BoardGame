### 缺失场景

#### 场景 1：持续能力 + 修正标记叠加

**描述**：持续能力修改影响力 + 修正标记修改影响力，两者叠加效果

**涉及卡牌**：
- Card06 占卜师（Diviner）：持续能力 - 强制对手先出牌
- Card08 审判官（Judge）：持续能力 - 赢得平局
- Card11 调停者（Mediator）：持续能力 - 强制平局
- Card12 财务官（Treasurer）：持续能力 - 额外印戒

**测试用例**：
1. 持续能力修改影响力 + 修正标记修改影响力 → 验证最终影响力计算正确
2. 多个持续能力同时生效 + 修正标记 → 验证优先级和叠加顺序
3. 持续能力移除后 + 修正标记仍存在 → 验证影响力重新计算

**优先级**：P1（核心组合场景）

---

#### 场景 2：多个持续能力同时生效

**描述**：多个持续能力同时生效时的优先级和叠加效果

**涉及卡牌**：
- Card06 占卜师 + Card08 审判官
- Card08 审判官 + Card11 调停者
- Card11 调停者 + Card12 财务官

**测试用例**：
1. 占卜师 + 审判官 → 验证先出牌顺序 + 平局处理
2. 审判官 + 调停者 → 验证优先级（已有测试但失败）
3. 调停者 + 财务官 → 验证平局 + 额外印戒

**优先级**：P1（核心组合场景）

**状态**：部分覆盖（`integration-ongoing-abilities.test.ts` 有测试但失败）

---

#### 场景 3：持续能力 + 印戒移动

**描述**：持续能力生效期间，印戒移动到其他卡牌

**涉及卡牌**：
- Card12 财务官（持续能力 - 额外印戒）
- Card13 傀儡师（Puppeteer）：替换卡牌，可能导致印戒转移

**测试用例**：
1. 财务官持续能力生效 → 傀儡师替换卡牌 → 验证印戒转移 + 持续能力是否仍生效
2. 持续能力附着在卡牌上 → 卡牌被替换 → 验证持续能力是否正确移除

**优先级**：P2（边界场景）

---

#### 场景 4：持续能力 + 卡牌弃掉

**描述**：持续能力生效期间，卡牌被弃掉

**涉及卡牌**：
- Card12 财务官（持续能力）
- Card01 雇佣剑士（Mercenary）：弃掉本牌和相对的牌

**测试用例**：
1. 财务官持续能力生效 → 雇佣剑士弃掉财务官 → 验证持续能力是否正确移除
2. 持续能力附着在对手卡牌上 → 对手卡牌被弃掉 → 验证持续能力是否正确移除

**优先级**：P2（边界场景）

## Task 12.3: 边界条件场景

### 缺失场景

#### 场景 1：弃牌能力 + 牌库为空

**描述**：弃牌能力在牌库为空时的行为

**涉及卡牌**：
- Card05 破坏者（Saboteur）：让对手弃掉牌库顶 2 张牌

**测试用例**：
1. 对手牌库为空 → 破坏者能力 → 验证不产生错误
2. 对手牌库只有 1 张牌 → 破坏者能力 → 验证只弃掉 1 张牌
3. 对手牌库为空 → 破坏者能力 → 验证能力是否被验证层拒绝（有效性门控）

**优先级**：P1（边界条件）

---

#### 场景 2：复制能力 + 无可复制目标

**描述**：复制能力在无可复制目标时的行为

**涉及卡牌**：
- Card09 女家庭教师（Governess）：复制能力

**测试用例**：
1. 场上无其他卡牌 → 女家庭教师能力 → 验证能力是否被验证层拒绝
2. 场上只有女家庭教师自己 → 女家庭教师能力 → 验证能力是否被验证层拒绝
3. 场上有卡牌但无可复制能力 → 女家庭教师能力 → 验证能力是否被验证层拒绝

**优先级**：P1（边界条件）

**状态**：`validation-effectiveness-gate.test.ts` 已覆盖部分场景

---

#### 场景 3：修正能力 + 场上无卡牌

**描述**：修正能力在场上无卡牌时的行为

**涉及卡牌**：
- Card02 图书管理员（Librarian）：添加修正标记
- Card15 虚空法师（Void Mage）：移除修正标记

**测试用例**：
1. 场上无卡牌 → 图书管理员能力 → 验证能力是否被验证层拒绝
2. 场上无修正标记 → 虚空法师能力 → 验证能力是否被验证层拒绝

**优先级**：P1（边界条件）

**状态**：`validation-effectiveness-gate.test.ts` 已覆盖部分场景

---

#### 场景 4：手牌为空 + 弃牌能力

**描述**：手牌为空时弃牌能力的行为

**涉及卡牌**：
- Card01 雇佣剑士（Mercenary）：弃掉本牌和相对的牌
- Card04 伏击者（Ambusher）：让对手弃掉指定派系手牌

**测试用例**：
1. 对手手牌为空 → 伏击者能力 → 验证不产生错误
2. 对手手牌为空 → 伏击者能力 → 验证能力是否被验证层拒绝（有效性门控）

**优先级**：P1（边界条件）

---

#### 场景 5：印戒数量边界

**描述**：印戒数量达到上限或下限时的行为

**涉及卡牌**：
- Card11 调停者（Mediator）：移除所有印戒
- Card12 财务官（Treasurer）：额外印戒

**测试用例**：
1. 卡牌印戒为 0 → 调停者能力 → 验证不产生错误
2. 卡牌印戒为 0 → 调停者能力 → 验证能力是否被验证层拒绝（有效性门控）
3. 卡牌印戒达到上限（如果有） → 财务官能力 → 验证印戒数量不超过上限

**优先级**：P2（边界条件）

---

#### 场景 6：遭遇结算边界

**描述**：遭遇结算时的边界条件

**涉及卡牌**：
- 所有卡牌

**测试用例**：
1. 双方影响力相等 → 验证平局处理
2. 双方影响力相等 + 审判官持续能力 → 验证审判官赢得平局
3. 双方影响力不等 + 调停者持续能力 → 验证调停者强制平局

**优先级**：P1（边界条件）

**状态**：`integration-ongoing-abilities.test.ts` 已覆盖部分场景（但失败）
