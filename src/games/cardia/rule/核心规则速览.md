# 《Duel For Cardia》核心规则速览（修正版）

## 一、游戏背景与目标

魔法之城卡迪亚由四大派系（绿图标反叛军、黄图标学者、红图标行会、蓝图标王朝）争夺统治权。

- **核心玩法**：每回合双方各打1张牌进行「遭遇」对抗。
- **胜利条件**：**第一个在自己场上所有卡牌的印戒总和达到5枚**的玩家获胜；建议采用三局两胜或五局三胜制。

## 二、游戏组件与开局设置

| 组件 | 数量与用途 |
| :--- | :--- |
| 角色牌 | 每位玩家16张（影响力1-16，分四大派系） |
| 地点牌 | 8张（新手局可忽略正面效果） |
| 印戒 | 12个（核心胜利道具） |
| 修正标记 | 22个（+1×8, -3×8, +5×6，用于修改影响力） |
| 持续标记 | 6个（用于持续效果，数量不限可用替代物） |
| 摘要牌 | 每位玩家2张（规则速查） |

**开局步骤：**
1. 将所有标记放于桌侧方便取用，地点牌洗混面朝下成堆，确定游戏方向。
2. 玩家各取16张同背面设计的I号角色牌，洗混作为自己的牌库，抽5张作为起始手牌。
3. 各拿2张摘要牌，准备开始。

## 三、回合完整流程（3阶段循环）

### 阶段1：打牌
- 双方从手牌各选1张，**面朝下**放置于场区，同时翻开比较影响力。
- **结算：**
  - 影响力高的牌：在**该牌上**放置**1枚印戒**。
  - 平局：无印戒，直接跳至阶段3。

### 阶段2：发动能力
- **只有输掉本次遭遇的牌**，其玩家才能发动该牌能力。
- **能力类型：**
  - **⚡ 即时能力**：立刻结算一次，效果结束。
  - **♾️ 持续能力**：在该牌上放持续标记，效果永久生效（即使该牌之后获胜）；标记移除则能力失效，需回溯结算所有影响。
- **关键规则：**
  - 能力效果优先级 > 通用规则。
  - 能力改变过去遭遇结果时，**过去输牌的能力不会重新触发**。
  - 修正影响力改变遭遇结果时，移动或归还印戒，**新输牌的能力不触发**。
  - **判定胜负前，必须先结算完所有场上能力效果**。

### 阶段3：回合结束
- 从自己牌库顶**抽1张牌**加入手牌（牌库空则不抽，不弃牌堆回库）。
- 检查胜利条件：
  - 若你场上所有卡牌的印戒总和≥5，立即获胜。
  - 若双方印戒总和均≥5且相等，继续游戏直到分出多少。

## 四、游戏结束与特殊情况

游戏立即结束并判定胜负的情况：
1. **阶段1**：对手无法出牌 → 你胜。
2. **阶段2**：结算的能力写明「你赢得游戏」 → 你胜。
3. **阶段3**：你场上所有卡牌的印戒总和≥5 → 你胜。

**平局/特殊胜负：**
- 若回合结束时双方印戒总和均≥5且相等 → 继续游戏，直到分出多少。
- 若阶段1双方都无法出牌 → 印戒总和多者胜；印戒总和相同则为平局。
- **优先级**：判定胜负前，必须先结算完所有场上能力效果。

## 五、核心概念速查

### 印戒机制
- 印戒放在**获胜的那张牌上**（不是玩家总池）
- 玩家的印戒总数 = 场上所有己方卡牌上的印戒之和
- 印戒可以因能力效果被移动或归还到补给区

### 能力触发规则
- 只有**当前遭遇中输掉的牌**才能发动能力
- 平局时跳过能力阶段
- 能力改变过去结果时，过去的能力不会重新触发
- 修正影响力改变结果时，新输掉的牌不触发能力

### 修正标记
- +1 影响力标记：8个
- -3 影响力标记：8个
- +5 影响力标记：6个
- 可叠加使用，最终影响力 = 基础值 + 所有修正标记

### 持续能力
- 需要放置持续标记（♾️）
- 效果持续生效，即使该牌之后获胜
- 标记移除时立即失效，需回溯结算所有影响
- 标记数量不限，可用替代物
