## 测试失败原因

测试失败是**业务逻辑问题**，不是框架迁移问题：

### 问题 1：渗透技能没有创建交互

**现象**：
```
⏳ 步骤 5: 等待交互出现
Test timeout of 60000ms exceeded.
```

**原因**：
- 打出渗透卡牌后，没有创建交互
- `sys.interaction.current` 一直是 `undefined`
- 等待交互超时（60秒）

**可能原因**：
1. 渗透技能的交互创建逻辑有问题
2. 交互系统配置不正确
3. 状态注入后交互没有正确触发

### 问题 2：服务器崩溃

**现象**：
```
Error: page.goto: net::ERR_CONNECTION_REFUSED at http://localhost:6173/play/smashup
```

**原因**：
- 第一个测试超时后，服务器崩溃
- 后续测试无法连接到服务器

**可能原因**：
1. 超时导致资源泄漏
2. 未捕获的异常导致服务器崩溃
3. 测试框架没有正确清理资源
