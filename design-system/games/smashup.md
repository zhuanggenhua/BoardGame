# Smash Up (大杀四方) 设计文档

## 核心美学方向：纸张混乱与地下室桌游之夜 (Paper Chaos & Basement Board Game Night)

### 1. 设计哲学 (Design Philosophy)
本项目版本的《大杀四方》不追求现代、扁平或数字化的界面。相反，它旨在模拟一种“物理感”——就像你正在朋友家的地下室，在一张老旧木桌上玩一局充满破坏力的漫画风格桌游。
- **物理性 (Physicality)**: 所有元素都应该是“物体”（纸张、贴纸、便签、木头、塑料筹码）。
- **非正式感 (Informal)**: 拒绝完美的对齐，利用轻微的选择和撕裂感来营造混乱但有趣的氛围。
- **美式漫画风格 (Comic Spoof)**: 利用大胆的粗边框、强烈的阴影和夸张的版式。

### 2. 视觉元素系统 (Visual Elements)

#### 背景 (Background)
- **材质**: 深色、温暖的木质纹理 (`#3e2723`)。
- **效果**: 强烈的暗角 (Vignette) 效果，将光线集中在桌子中心的对战区域。
- **纹理**: 叠加轻微的杂质和木纹。

#### 卡牌表现 (Card Presentation)
- **白边 (White Border)**: 所有卡牌都有明显的宽白边，看起来像刚剪下来的贴纸。
- **随机旋转 (Random Rotation)**: 在手牌、牌堆或基地上的卡牌都有轻微的随机旋转角度（-2° 到 2°），模拟自然摆放。
- **阴影**: 较大的软阴影，营造卡牌堆叠在桌面上的深度感。

#### UI 容器 (UI Containers)
- **撕纸效果 (Torn Paper)**: 所有的面板（如等待条、进度条）使用 `clip-path-jagged` 实现手动撕裂的边缘感。
- **黄色便签 (Sticky Notes)**: 状态指示器、回合提醒采用典型的黄色便签（#fef3c7）样式，配有胶带或图钉装饰。
- **记分册 (Score Sheet)**: 右上角的记分板设计成老式记分纸样式，带有物理性的塑料“筹码”代表玩家角色。

### 3. 色彩系统 (Color Palette)
- **主背景**: 木纹色 (`#3e2723`)
- **UI 强调色**: 黄便签色 (`#fef3c7`, `#ffd54f`)
- **状态高亮**: 漫画风格的青色 (`#22d3ee`) 或 亮橙色 (`#f59e0b`)
- **阴影**: 沉重的黑色 (`rgba(0,0,0,0.6)`)，而非柔和的灰色。

### 4. 交互规范 (Interaction)
- **禁止毛玻璃**: 为了保持纸张和实体的质感，且为了在高频率交互中保持性能，严禁使用 `backdrop-blur`。
- **动画**: 
  - 抽牌/出牌应有明显的物理位移。
  - 按钮互动应有明显的“按下”深度感。
  - 尽量使用 framer-motion 的 spring 物理引擎来实现自然的弹性。

### 5. 性能守则 (Performance Rules)
- 鉴于玩家需要频繁查看卡牌大图，遮罩层应使用简单的半透明黑色覆盖，避免任何昂贵的后处理滤镜，确保“看牌-合牌”的操作瞬间完成。
