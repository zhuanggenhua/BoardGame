# DiceThrone (王权骰铸) UI 设计

> **基于风格**：`design-system/styles/arcade-3d.md`
> **主色调**：琥珀金

---

## 风格选择理由

- 骰子类游戏，需要"掷骰"的物理感和爽快感
- 派对向，适合活泼大胆的视觉风格
- 技能卡牌需要醒目的交互反馈

---

## 专属配置

### 主色调

使用 `arcade-3d.md` 中的**琥珀金**预设：

```css
--primary-light: #fcd34d;   /* amber-300 */
--primary-from: #fbbf24;    /* amber-400 */
--primary-to: #d97706;      /* amber-600 */
--primary-shadow: #b45309;  /* amber-700 */
```

### 玩家颜色

| 玩家 | 主色 | 用途 |
|------|------|------|
| 玩家 1 | 蓝色系 | 头像边框、回合指示 |
| 玩家 2 | 红色系 | 头像边框、回合指示 |

### 专属组件

#### 骰子
- 使用 3D 旋转动画
- 锁定状态有金色边框 + 光晕
- 可选中状态有脉冲动画

#### 技能卡
- 使用游戏专属边框图片
- Hover 时轻微上浮 + 发光
- 禁用时灰度 + 降低透明度

#### 英雄面板
- 使用游戏专属背景图
- 血量条使用红色渐变
- CP 条使用蓝色渐变

---

## 覆盖规则

以下情况覆盖基础风格：

1. **技能卡边框**：使用游戏资源中的专属边框，而非通用圆角
2. **骰子样式**：使用 3D 骰子组件，而非 2D 图标
3. **英雄选择界面**：使用全屏沉浸式布局
